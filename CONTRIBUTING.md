# Contributing to Agent Memory System

Thanks for your interest in contributing!

## Quick Start

1. Fork the repo
2. Create a feature branch: `git checkout -b my-feature`
3. Make your changes
4. Test: `bash scripts/test-runner.sh`
5. Submit a PR

## Ways to Contribute

- **Bug fixes** - Check [Issues](https://github.com/Xzeroone/agent-memory-system/issues)
- **New features** - Propose in Discussions first
- **Integrations** - Add support for new frameworks (LangChain, CrewAI, etc.)
- **Documentation** - Improve guides, add examples
- **Testing** - Add test cases for edge conditions

## Code Style

- Bash: Follow [Google Shell Style Guide](https://google.github.io/styleguide/shellguide.html)
- Python: PEP 8, type hints preferred
- Comments: Explain "why", not "what"

## Pull Request Process

1. Update relevant documentation
2. Add tests if applicable
3. Keep PRs focused (one feature/fix per PR)
4. Link related issues
